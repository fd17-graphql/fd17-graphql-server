> npm start

## MongoDB
### _start MongoDB_
```
> cd fd17-graphql-server
> ./mongod
```


#### _mongoshell (s. also https://docs.mongodb.com/getting-started/shell/query/)_
```
> mongo
```


#### _mongoimport_
```
> cd fd17-graphql-server
> mongoimport --db fd17 --collection partner --file /db/insert-scripts/insert-fd17-partner.json
```
